## 🔧 Arduino Uno (Голова)

```
         ┌────────────────────────┐
         │     ARDUINO UNO        │
         │       (CH340)          │
         │                        │
     D3  │ ●  ── 220Ω ── RGB R   │
     D5  │ ●  ── 220Ω ── RGB G   │
     D6  │ ●  ── 220Ω ── RGB B   │
     D4  │ ●  ── Servo eyeL      │ ← Левый глаз ↔
     D7  │ ●  ── Servo eyeR      │ ← Правый глаз ↔
     D8  │ ●  ── Servo jaw       │ ← Челюсть ↕
     D9  │ ●  ── Servo headRot   │ ← Поворот головы ↔
     D10 │ ●  ── Servo neckPitch │ ← Шея ↕ (наклон)
     D11 │ ●  ── Servo neckYaw   │ ← Шея ↔ (поворот)
         │                        │
     GND │ ●  ── RGB GND + Servo GND (общий) │
     5V  │ ●  ── RGB VCC + Servo VCC         │
         │                        │
     USB │ ──── Raspberry Pi USB1  │
         └────────────────────────┘
```

### Детальная таблица пинов

| Пин | Цвет провода | Компонент | Куда подключать |
|-----|:---:|-----------|-----------------|
| D3 | 🔴 Красный | RGB R → 220Ω | LED кращёный анод |
| D5 | 🟢 Зелёный | RGB G → 220Ω | LED зелёный анод |
| D6 | 🔵 Синий | RGB B → 220Ω | LED синий анод |
| D4 | Оранжевый | Servo eyeL | Сигнал левого глаза |
| D7 | 🟡 Жёлтый | Servo eyeR | Сигнал правого глаза |
| D8 | Белый | Servo jaw | Сигнал челюсти |
| D9 | 🟣 Фиолетовый | Servo headRot | Сигнал поворота головы |
| D10 | Серый | Servo neckPitch | Сигнал наклона шеи |
| D11 | 🟤 Коричневый | Servo neckYaw | Сигнал поворота шеи |
| GND | ⬛ Чёрный | GND (общий) | LED GND + все Servo GND + внешний блок GND |
| 5V | 🔴 Красный | VCC | LED + Servos (⚠️ от Arduino 5V только RGB) |

### RGB LED (4-pin, общий катод)

| Пин LED | Через резистор | Подключать к |
|:---:|:---:|:---:|
| R (длинный) | 220Ω | Arduino D3 |
| G | 220Ω | Arduino D5 |
| B | 220Ω | Arduino D6 |
| GND (короткий) | — | Arduino GND |

### Сервоприводы (3 провода каждый)

| Цвет | Значение |
|:---:|------|
| 🔴/🟠 Красный/оранжевый | VCC (5V) |
| ⬛/🟤 Чёрный/коричневый | GND |
| 🟡/🟡 Жёлтый/белый | Signal → Arduino pin |

## ⚡ Питание

```
                    ┌─────────────┐
                    │  Сеть 220V  │
                    └──────┬──────┘
                           │
          ┌────────────────┼────────────────┐
          │                │                │
    ┌─────▼─────┐   ┌─────▼─────┐   ┌─────▼─────┐
    │ Pi 5V 2.5A│   │ Блок 5-6V │   │ Блок 5-6V │
    │ (микроUSB)│   │   2A+     │   │   2A+     │
    └─────┬─────┘   └─────┬─────┘   └─────┬─────┘
          │                │                │
          │           ┌────┴────┐      ┌────┴────┐
          │           │  Servos │      │  Servos │
     ┌────▼────┐      │  (Mega) │      │  (Uno)  │
     │ Raspberry│      └────┬────┘      └────┬────┘
     │    Pi   │           │                │
     │         │◄──USB0───►│                │
     │         │           │                │
     │         │◄──USB1────────────────────►│
     └─────────┘           │                │
          │                │                │
          ▼                ▼                ▼
         GND ──────────── GND ──────────── GND
              (соединить все GND вместе!)
```

### ⚠️ ПРАВИЛА ПИТАНИЯ

| Правило | Почему |
|--------|--------|
| ❌ НЕ питать серво от Arduino 5V | Arduino отдаёт только 500mA, серво потребляют 2A+ |
| ✅ Отдельный блок 5-6V на каждый блок серво | Стабильное напряжение, не перегружает Arduino |
| ✅ Соединить все GND вместе | Общая точка отсчёта для сигналов |
| ✅ Конденсатор 100µF на линию питания серво | Сглаживание просадок |

## 📡 Serial команды (быстрый справочник)

### Uno (Голова) — `/dev/ttyUSB0`

| Действие | Команда |
|----------|---------|
| Левый глаз влево | `SERVO:eyeL:45` |
| Левый глаз центр | `SERVO:eyeL:90` |
| Левый глаз вправо | `SERVO:eyeL:135` |
| Правый глаз влево | `SERVO:eyeR:45` |
| Правый глаз центр | `SERVO:eyeR:90` |
| Правый глаз вправо | `SERVO:eyeR:135` |
| Челюсть закрыть | `SERVO:jaw:90` |
| Челюсть открыть | `SERVO:jaw:45` |
| Челюсть широко | `SERVO:jaw:135` |
| Голова прямо | `SERVO:headRot:90` |
| Голова влево | `SERVO:headRot:45` |
| Голова вправо | `SERVO:headRot:135` |
| Шея вперёд | `SERVO:neckPitch:45` |
| Шея назад | `SERVO:neckPitch:135` |
| Шея повернуть Л | `SERVO:neckYaw:45` |
| Шея повернуть П | `SERVO:neckYaw:135` |
| RGB красный | `RGB:255,0,0` |
| RGB зелёный | `RGB:0,255,0` |
| RGB синий | `RGB:0,0,255` |
| RGB белый | `RGB:255,255,255` |
| RGB выкл | `OFF` |
| RGB пресет | `PRESET:red` |
| Статус | `STATUS` |

| Правое плечо вниз | `SERVO:shoulderR:135` |
| Левый локоть согнуть | `SERVO:elbowL:45` |

| Правый локоть разогнуть | `SERVO:elbowR:135` |

## 🔧 Чеклист подключения

- [ ] 1. Подключить GND внешнего блока питания к GND Arduino
- [ ] 2. Подключить VCC внешнего блока (5-6V) к проводам VCC серво
- [ ] 3. Подключить сигнальные провода серво к пинам Arduino (по таблице)
- [ ] 4. Подключить RGB LED: R→D3, G→D5, B→D6 (через 220Ω), GND→GND
- [ ] 5. Подключить Arduino Uno к Pi USB1 (/dev/ttyUSB0)
- [ ] 6. Подключить Arduino Mega к Pi USB0 (/dev/ttyACM0)
- [ ] 7. Подключить Raspberry Pi к питанию 5V 2.5A
- [ ] 8. Проверить: `ls /dev/ttyUSB* /dev/ttyACM*`
- [ ] 9. Загрузить скетчи (см. README → Загрузка скетчей)
- [ ] 10. Запустить: `cd ~/robot-catty/server && npm start`
- [ ] 11. Открыть: `http://<pi-ip>:3000`
