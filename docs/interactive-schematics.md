## Описание компонентов

### Arduino Uno (Голова)

| Компонент | Пин | Описание |
|-----------|-----|----------|
| RGB LED R | D3 | Красный канал через 220Ω резистор |
| RGB LED G | D5 | Зелёный канал через 220Ω резистор |
| RGB LED B | D6 | Синий канал через 220Ω резистор |
| Servo L | D4 | Левый глаз (0-180°) |
| Servo R | D7 | Правый глаз (0-180°) |
| Servo J | D8 | Челюсть (90° закрыта, 135° открыта) |

### Arduino Mega (Тело)

| Компонент | Пин | Описание |
|-----------|-----|----------|
| Shoulder L | D2 | Левое плечо |
| Shoulder R | D3 | Правое плечо |
| Elbow L | D4 | Левый локоть |
| Elbow R | D5 | Правый локоть |

### Raspberry Pi 3B

| Функция | Подключение |
|---------|-------------|
| Arduino Uno | USB → /dev/ttyUSB0 |
| Arduino Mega | USB → /dev/ttyACM0 |
| Сеть | Ethernet / WiFi |
| Питание | 5V 2.5A (отдельный блок) |

## Питание

⚠️ **КРИТИЧНО**: Сервоприводы НЕ питать от Arduino 5V!

- **Arduino Uno/Mega**: питание от USB Raspberry Pi
- **Сервоприводы (7шт)**: отдельный блок 5-6V, минимум 2A на каждый серво
- **RGB LED**: питание от Arduino через резисторы 220Ω
- **GND**: соединить GND всех источников питания вместе

## Serial Protocol (9600 baud)

### Arduino Uno (Голова)

| Команда | Описание | Пример |
|---------|----------|--------|
| `SERVO:<name>:<angle>` | Установить угол серво (0-180) | `SERVO:eyeL:120` |
| `RGB:<r>,<g>,<b>` | Установить цвет (0-255) | `RGB:255,128,0` |
| `OFF` | Выключить RGB | `OFF` |
| `FADE:<r>,<g>,<b>` | Плавный переход к цвету | `FADE:0,0,255` |
| `BLINK` | Режим мигания | `BLINK` |
| `PRESET:<name>` | Готовый цвет | `PRESET:red` |
| `STATUS` | Текущее состояние | `STATUS` |

### Arduino Mega (Тело)

| Команда | Описание | Пример |
|---------|----------|--------|
| `SERVO:<name>:<angle>` | Установить угол серво (0-180) | `SERVO:shoulderL:120` |
| `CENTER` | Центрировать все серво | `CENTER` |
| `STATUS` | Текущее состояние | `STATUS` |
