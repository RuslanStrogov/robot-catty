## Описание компонентов

### Arduino Uno (Голова)

| Компонент | Пин | Описание |
|-----------|-----|----------|
| RGB LED R | D3 | Красный канал через 220Ω резистор |
| RGB LED G | D5 | Зелёный канал через 220Ω резистор |
| RGB LED B | D6 | Синий канал через 220Ω резистор |
| Servo L | D4 | Левый глаз (0-180°) |
| Servo R | D7 | Правый глаз (0-180°) |
| Servo J | D8 | Челюсть (90° закрыта, 135° открыта) |
| Servo | D9 | Поворот головы ↔ |
| Servo | D10 | Шея ↕ (наклон) |
| Servo | D11 | Шея ↔ (поворот) |

### Arduino Mega (Тело)

| Компонент | Пин | Описание |
|-----------|-----|----------|
| Shoulder L | D2 | Левое плечо |
| Shoulder R | D3 | Правое плечо |
| Elbow L | D4 | Левый локоть |
| Elbow R | D5 | Правый локотель |

### Raspberry Pi 3B

| Функция | Подключение |
|---------|-------------|
| Arduino Uno | USB → /dev/ttyUSB0 |
| Arduino Mega | USB → /dev/ttyACM0 |
| Сеть | Ethernet / WiFi |
| Питание | 5V 2.5A (отдельный блок) |
